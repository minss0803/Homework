# Homework

## 개발 환경
- minVersion : iOS 10
- targetVersion : iOS 13
- Swift 5
- Xcode 11
- CocoaPods

## 실행 방법
```
1. git clone https://github.com/minss0803/Homework.git
2. pod install
3. open Homework.xcworkspace
4. Run the application
```

## 구현 방법
- Swift 및 그 외 Dependency를 사용했습니다.
- MVP 패턴으로 구현하였습니다.
